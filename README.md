# katalekto
Software para mejorar la fluided lectora en euskera y castellano.
Programado en Genie/Vala

Se trata de un programa que muestra textos de una forma controlada.

Puede mostrar textos de parrafo en parrafo, haciendo aparecer palabras progresivamente o haciendolas desaparecer,
o también mostrando las palabras del texto de una en una.

La mejora con respecto a Katamotz Lectura es que tiene la capacidad de editar textos. También puede crear actividades 
en base a unas especificaciones facilmente configurables en varios idiomas: Euskera y Castellano.

Katalekto puede configurar la velocidad de lectura y también el tamaño de fuente usado.
